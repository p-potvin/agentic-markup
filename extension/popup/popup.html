Active on this page

Markup Syntax

  • :::collapse[Title] ... ::: Collapsible section
  • :::callout[info|warning|error|success] ... ::: Callout box
  • :::tabs[Tab 1|Tab 2] content\n---\ncontent ::: Tabbed panel
  • :::badge[text][variant]::: Inline badge

Ask the AI assistant to wrap its response in one of the markers above to get a rich interactive component.
